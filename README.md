# 💖 A Special Romantic Proposal Web App

A beautiful, interactive 4-step progressive web application designed with fluid animations, custom-themed components, and dynamic UI logic. 

## 🚀 Live Demo
You can view the live project here: *[(https://saurav56295.github.io/special-entry/)]*

---

## ✨ Features
* *Progressive Node Tracker:* A custom status timeline bar tracking user interactions from introduction to the ultimate choice.
* *Smart Interaction Controls:* Built-in verification routing that requires specific inputs to unlock subsequent questions.
* *Dynamic Escape Button:* A "No" button logic system that calculationally moves away from cursor hovering.
* *Immersive Celebration Layer:* Seamless full-screen scaling presentation featuring a parallax background illustration and falling particle effects upon user acceptance.

---

## 🛠️ Tech Stack Used
* *HTML5:* Semantic architecture layout structures.
* *CSS3:* Custom linear glassmorphic gradients, UI component layouts, responsive layout adaptation viewports, and interactive keyframe animation trees.
* *JavaScript (ES6):* Dynamic step tracking logic, multi-stage state management, random positioning calculations, and item rendering cycles.
